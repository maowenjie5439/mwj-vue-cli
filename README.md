mwj-vue-cli
📖简介
用于快速搭建前端项目的命令行工具
📕 模板
✅ 快速生成 Vue3+TS+Vite+ElementPlus+Pinia+Axios+Mock的模板
📦 安装
npm install mwj-vue-cli -g
🚩使用
# 使用模板创建新项目
mwj-vue-cli create 

# 执行项目文件夹名称并选择模板创建新项目
mwj-vue-cli create [projectName]

# 更新脚手架到最新版本
mwj-vue-cli update

# 查看脚手架版本
mwj-vue-cli -v
