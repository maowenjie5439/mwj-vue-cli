从0到1构建脚手架

1. git init

2. npm init: 创建package.json文件

3. vue-tsc --init: 创建tsconfig.json文件

   > vue-tsc 是专门为 Vue 3 项目设计的 TypeScript 类型检查工具。
   >
   > ### 主要功能
   >
   > 类型检查
   >
   > ​	检查 .vue 文件中的 TypeScript 代码
   >
   > ​	检查模板语法中的类型错误
   >
   > ​	验证 props、emits 等 Vue 特定功能的类型
   >
   > 命令行工具
   >
   > ​	可以在构建过程中进行类型检查
   >
   > ​	支持与 VS Code 等编辑器集成，提供实时类型检查
